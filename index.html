- A menu page that displays the different coffee varieties and prices.
- A cart page that allows customers to view their selected items and update their order before submitting it.

Farm Advisor

Web App

Farm Advisor is a web application designed to help farmers make informed
decisions about their crops and livestock.
The app provides information on weather conditions, soil types, and crop types to help farmers optimize their yield.
The app also provides recommendations for fertilizers and pesticides based on the specific crop and soil conditions.
Farm Advisor is designed to be user-friendly and accessible, with a simple interface that can be used on any device with an internet connection.
The app is built using modern web technologies, including React for the front-end and Node.js for the back-end, ensuring scalability and robustness.
